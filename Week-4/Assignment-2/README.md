# How to test my code?

### 1. Install the required package:
```
$ npm install 
```

### 2. Change the value in .env to your own value:
```
PORT = "your_port"
```

### 3. Run the app.js:
```js
$ node app.js  // or $ nodemon app.js
```

### 4. Input the url in your browser and check the page:
[http://localhost:3000/](http://localhost:3000/)
